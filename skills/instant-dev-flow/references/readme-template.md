## Quick Start

### Prerequisites

- Node.js >= 20
- npm or pnpm
- [Database setup instructions if applicable]

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/[org]/[repo].git
   cd [repo]
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   ```bash
   cp .env.example .env.local
   # Edit .env.local with your values
   ```

4. Run database migrations (if applicable):
   ```bash
   # [Migration command]
   ```

5. Start the development server:
   ```bash
   npm run dev
   ```

6. Open [http://localhost:3000](http://localhost:3000)

## Writing Rules

1. **Introduction**: Focus on "what" and "why", not "how"
2. **Key Features**: 3-6 features, each with a bold name and one-line description
3. **Tech Stack**: Only list technologies actually used
4. **Project Structure**: Show the most important directories (max 3 levels deep)
5. **Documentation**: Link to all docs generated by the workflow
6. **Quick Start**: Minimal steps to get from clone to running
7. **Keep it concise**: README should be scannable in under 2 minutes
